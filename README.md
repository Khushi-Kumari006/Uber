#  Uber Website UI Clone

##  Project Overview

The **Uber Website UI Clone** is a static front-end project developed using **HTML5** and **CSS3**.
The objective of this project is to replicate the visual design and layout of the official **Uber** homepage.

This project focuses entirely on user interface development without implementing backend functionality or JavaScript-based interactivity.

It demonstrates the ability to recreate a real-world website layout using structured HTML and modern CSS styling techniques.

The clone closely matches the original Uber homepage in terms of layout, typography, spacing, color scheme, and responsiveness.

---

##  Purpose of the Project

The primary purpose of this project is to strengthen frontend development skills by:

* Recreating a real-world website layout
* Practicing HTML page structuring
* Improving CSS styling techniques
* Understanding responsive design principles
* Building portfolio-ready projects

This project serves as a practical implementation of core frontend concepts.

---

##  Features Implemented

### 🔹 Navigation Bar

* Structured top navigation similar to the Uber homepage
* Proper alignment using Flexbox
* Styled buttons and links

### 🔹 Hero Section

* Large banner section
* Headline and call-to-action buttons
* Proper spacing and alignment

### 🔹 Service Sections

* Multiple content sections structured to resemble Uber’s layout
* Use of images and text alignment
* Clean spacing and grid-based layout

### 🔹 Footer Section

* Structured footer layout
* Organized links and informational sections

### 🔹 Responsive Design

* Layout adapts to different screen sizes
* Use of CSS Flexbox and Grid
* Media queries for responsiveness

```

```

### Step 2: Navigate to the project folder

```
cd Uber-Clone
```

### Step 3: Open the project

Open `uber.html` in any modern web browser.

No additional setup or dependencies are required.
