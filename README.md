# Uber Clone (HTML & CSS)
A responsive front-end clone of the Uber landing page built using pure HTML5 and CSS3.
This project focuses on recreating a real-world UI to strengthen core frontend fundamentals like layout structuring, positioning, and responsive design.

✨ Features
Fully responsive layout (mobile → desktop)
Clean and modern UI inspired by Uber
Flexbox and CSS Grid based layout
Smooth hover effects and transitions
Cross-browser compatible design
Well structured and readable code
🛠️ Tech Stack
HTML5 (Semantic markup)
CSS3 (Flexbox, Grid, Media Queries)
Google Fonts
Font Awesome Icons
📂 Project Structure
uber-clone/ ├── uber.html ├── uber.css └── README.md

🎯 Sections Implemented
Navigation bar
Hero booking section
Ride suggestion cards
Schedule ride section
Ride sharing information
Footer
